# LLAgents Web

Next.js app for user registration and template deployment.

## Development

```bash
npm install
npm run prisma:push
npm run dev
```

## Environment

```text
MANAGER_URL=http://manager:8080
APP_ROOT_DOMAIN=llmagents.com
DATABASE_URL=mysql://root:password@mysql:3306/llagents_platform
AUTH_SECRET=change-me
```
